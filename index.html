❄️
1
Dolang Skills 공식 사이트 접속 (1순위)
dolangskills.com → 상단 메뉴 Products → HVAC Training 클릭

2
Alibaba 동시 검색 (가격 비교용)
alibaba.com 접속 후 검색어 입력:
"portable HVAC trainer kit CE OEM"
"refrigeration trainer vocational education"
필터: Supplier type → Verified Supplier / Min. Order → 1 unit / Certification → CE
3
공급사 필터링 기준
✓ CE 인증서 번호 확인 (EU EUDAMED 조회)
✓ ISO 9001 유효기간 확인
✓ Trade Assurance 공급사 우선 선택
✓ 영문 매뉴얼·부품 도면 제공 여부 확인
4
샘플 발주 → OEM 협상
MOQ 1 set 샘플 발주 → 품질 확인 후 OEM 협상
OEM 요청 사항: IEG 로고 레이저각인 + 전용 박스 패키징
결제 조건: 30% 선불 / 70% 선적 전 T/T
목표 원가: $120–200/unit (판매가 $280–420)

→ PCB 설계 외주: 한국 내 PCB 제조사 (코리아서킷 등)
→ 이 보드가 복제 불가 경쟁력 = Lock-in 핵심
4
발주 전 최종 체크
✓ 24VDC I/O 호환 여부 (모든 PLC 브랜드 연결 가능)
✓ 영문 래더 로직 예제 프로그램 포함 여부
✓ 부품 단독 교체 가능 (모듈형 구조) 확인
