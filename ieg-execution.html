Core Demo · 6:00–30:00
실습 본편 — 래더 다이어그램 작성 & 시뮬레이션
24분 | ⭐ TikTok 클립 구간 (29:45–30:00 — 실장비 구동)

[단계 1 · 6:00–12:00] 새 프로젝트 생성

  • GX Works3 실행 → New Project → QCPU 선택
  • "Q03UDV" 모델 선택 — "이게 우리 장비에 들어 있는 실제 CPU입니다"
  • Simple Project → Ladder 선택
  • → 화면 녹화 + 강사 손 클로즈업 교차편집

[단계 2 · 12:00–20:00] 래더 작성

  • X0 (START 버튼) — NO 접점 입력
  • X1 (STOP 버튼) — NC 접점 입력
  • Y0 (컨베이어 모터) — 코일 출력
  • Y0 자기유지 회로 추가 (Seal-in contact)
  • "자기유지가 없으면 버튼에서 손을 떼는 순간 모터가 꺼집니다 — 이게 래더의 핵심 개념입니다"

[단계 3 · 20:00–28:00] 컴파일 & 시뮬레이션

  • Convert → Compile → 오류 없음 확인
  • Debug → Simulation 모드 진입
  • X0 강제 ON → Y0 점등 확인 (자기유지 동작)
  • X1 ON → Y0 소등 확인
  • "실제 장비에 다운로드하면 이게 컨베이어를 움직입니다"

[단계 4 · 28:00–30:00] 실장비 적용 데모 ⭐ 핵심 클립

  • USB 케이블로 PLC 패널에 연결
  • Write to PLC → 프로그램 다운로드
  • 실제 버튼 조작 → 컨베이어 벨트 구동
  • → 이 15초 클립이 TikTok 메인 훅 영상이 됩니다. 반드시 클로즈업 별도 촬영.
